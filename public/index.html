Workflow guidance

  1. Tap a waypoint on the map to load the corresponding first-person image.
  2. Use the route planner to highlight an optimal path between two points.
  3. When you capture additional rooms or wings, append them to public/data/waypoints.json and drop new imagery into assets/images.
  4. Regenerate the layout using scripts/update_waypoints.py (coming soon) or edit the JSON manually with precise coordinates.
